# Bioclimatic Zone Categorical Coverage

This block defines a semantic CoverageJSON profile for categorical bioclimatic zones. Categories are exposed through `categories` and `categoryEncoding`, while the grid values are stored in the `ranges` object.

The included example is aligned with the Finland VAP grid and is intended as a demonstrator layer. For scientific use, replace it with an official bioclimatic zonation coverage while preserving the same semantic structure.
